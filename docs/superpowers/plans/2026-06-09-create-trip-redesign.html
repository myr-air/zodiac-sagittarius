Implementation Plan: Create Trip Redesign

Goal: replace the current decorative boarding-pass create-trip page with a calm operational setup console that helps organizers enter trip basics, compare destinations, set dates, and understand invite readiness without broken text or giant blank regions.

Design lane: restrained product UI. Anchor references: Stripe Dashboard creation with useful live preview, GitHub new repository setup simplicity, Linear issue creation focus, TripIt manual trip data scope, and Wanderlog group-planning/map scope.
